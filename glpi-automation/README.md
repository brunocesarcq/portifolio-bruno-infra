# Automação de Atualização do GLPI

Scripts para atualização segura do GLPI em ambientes Linux produtivos.

## Objetivo
Garantir atualização previsível, com backup e validação pós-upgrade.

## Tecnologias
- Bash
- Linux
- Apache / PHP
- MySQL / MariaDB
- GLPI

## Fluxo
1. Backup
2. Atualização
3. Validação

## Boas Práticas
- Backup obrigatório
- Processo documentado
- Execução controlada
